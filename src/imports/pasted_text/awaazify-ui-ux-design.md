Design a modern, clean, scalable, and hackathon-winning web app UI/UX for a civic intelligence platform called:

"Awaazify – Raise Your Voice"

IMPORTANT:
Focus on STRUCTURE, FLOW, and UX hierarchy inspired by the Green Points homepage layout (section-based storytelling, spacing, and flow), but DO NOT copy colors. Use a fresh, professional civic-tech design.

The UI must feel like a REAL PRODUCT, not a student project.

-------------------------------------
🔐 AUTHENTICATION SYSTEM (COMPLETE FLOW)
-------------------------------------

Design a full authentication experience:

1. Login / Sign Up (Tab or Split Layout)
   - Email + Password
   - Google Sign-In (PRIMARY CTA)
   - Phone Number Login (OTP-based)
   - "Remember Me"
   - Forgot Password

2. Multi-Step Sign Up:
   Step 1: Name, Email, Password
   Step 2: Phone OTP Verification (required for posting)
   Step 3: Profile Setup (Location, Interests, Issue Categories)

3. Verified User System:
   - Verified badge UI
   - Only verified users can post issues
   - Show trust level indicator

4. Social Media Integration:
   - Connect LinkedIn, Twitter (X), Facebook
   - Toggle: "Auto-share my posts"
   - After posting → "Share this issue" modal

5. UI Style:
   - Card-based
   - Clean inputs with icons
   - Step progress indicator

-------------------------------------
🚨 CORE PRODUCT LOGIC UI (VERY IMPORTANT)
-------------------------------------

Design UI elements that reflect intelligence:

1. Priority Score Display:
   - Show score badge on issue cards
   - Label: High / Medium / Critical

2. Severity Selector (NEW):
   - While posting: Low / Medium / High / Critical
   - Affect visual emphasis (color/indicator)

3. Smart Duplicate Detection:
   - Before posting → show:
     "Similar issue already exists. Join instead?"
   - Show list of similar issues

4. Trust & Anti-Spam System:
   - Show user trust score
   - Report abuse button
   - Limit multiple "Me Too"
